## Workstation Setup

### Programs I recommend:

- [TreeSize](https://www.jam-software.com/treesize): Modern version of WinDirStat, runs very fast and has useful file operations.
- [UltraSearch](https://www.jam-software.com/ultrasearch): A fast file search utility for Windows with advanced filtering and indexing capabilities.
- Docker Desktop: Windows Desktop version of the Docker engine, used instead of installing dev environments onto my system.
- CPU-Z & GPU-Z: Useful system profiling and info.
- WiseToys: Useful Plugins for Windows
- Windows PowerToys & Power Automate: Various tools and features for Windows power users like myself.


My operating system for my Desktop is Windows 11. I use [Visual Studio Code](https://code.visualstudio.com/) as my IDE for all code editing.
